# AI-Driven Smart Fire Detection System

This project detects potential fire risks using IoT sensors and machine learning.

## Features

- Real-time sensor monitoring
- AI fire prediction model
- Web dashboard
- Emergency alarm system

## Hardware

- Arduino UNO / ESP32
- Flame Sensor
- MQ2 Smoke Sensor
- DHT11 Temperature Sensor
- Buzzer

## Technologies

- Arduino
- Python
- Machine Learning
- HTML CSS JavaScript
